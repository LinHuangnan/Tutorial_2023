名称
    stu.sh

摘要
    帮助你统计在册学生

语法
    ./stu.sh [ -a ] [序号:人名:性别:专业/职位] 
    ./stu.sh [ -s | -d ] [序号] 

说明
    [],表示可选
        如果单独输入./stu.sh，它会输出所有在册学生

    -a,代表了append——添加
        如果你输入的学生信息已经在册，它就会提示你这个情况
        如果你输入的学生信息并不在册，它就会自动帮你添加上去

    -s,代表了search——搜索
        如果你输入的学生信息已经在册，它就会帮你单独搜索出来
        如果你输入的学生信息并不在册，它就会提示你这个情况
    
    -d,代表了delete——删除
        如果你输入的学生信息已经在册，它就会帮你单独搜索出来
        如果你输入的学生信息并不在册，它就会提示你这个情况

例子
    已知张三的序号，想要查询张三的信息
    >> ./stu.sh -s 100

    已知张三的序号，想要删除张三的信息
    >> ./stu.sh -d 100

    已知张三的所有信息，想要帮张三登记在册
    >> ./stu.sh -a 100:张三:男:计算机
